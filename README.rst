ProsperTradier
==============

|Build Status| |Coverage Status| |PyPI Badge|

Python library for `Tradier's Developer REST API`_.  

Getting Started
===============

	``pip install .``

Running tests is easy!

	``python setup.py test``

Notes
-----

- Only supporting Python3 at this time
